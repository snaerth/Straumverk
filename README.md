## Website

[Straumverk webite](https://www.straumverk.is)

## Development

```
npm install
npm run dev
```

or with yarn

```
yarn
yarn dev
```

## Production

```
npm run build
npm run start
```

or with yarn

```
yarn build
yarn start
```

## Production server

```
npm run build
npm run pm2
```

or with yarn

```
yarn build
yarn pm2
```
